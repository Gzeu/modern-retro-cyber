🚀 Modern Retro Cyber Documentation

🌟 Welcome to the future of Web3 gaming development!
This documentation will guide you through everything you need to know to build amazing crypto games.

Modern Retro Cyber is a production-ready monorepo template that combines the best of modern web development with cutting-edge Web3 technology. Whether you're building the next viral crypto game or just learning Web3 development, this framework has everything you need.

⚡ Quick Installation

# Magic setup - everything configured in 60 seconds curl -fsSL https://raw.githubusercontent.com/Gzeu/modern-retro-cyber/main/scripts/install.sh | bash # Or clone manually git clone https://github.com/Gzeu/modern-retro-cyber.git cd modern-retro-cyber pnpm install && pnpm dev

🏢 Architecture Overview

Our monorepo is structured for maximum developer productivity and maintainability:

modern-retro-cyber/ ├── apps/ │ ├── web/ # Main Remix application │ └── admin/ # Admin dashboard ├── packages/ │ ├── ui/ # React component library │ ├── config/ # Shared configurations │ └── tsconfig/ # TypeScript configurations ├── contracts/ # Smart contracts (Hardhat) └── .github/workflows/ # CI/CD automation

🎨 UI Components

Our UI library includes gaming-optimized React components:

import { Button, Card, Badge, Brand, TokenMinerGame } from '@modern-retro-cyber/ui'; // Gaming-first components <TokenMinerGame onMint={(tokens) => mintToWallet(tokens)} theme="retro-cyber" animations="smooth" />

🎮 Gaming Features

Built-in gaming mechanics that make Web3 games engaging:

  • 🎮 Interactive Mining: Click-to-earn mechanics with visual feedback
  • 📈 Level Progression: XP system with increasing mining rates
  • 🔄 Auto-Mining: Passive income generation
  • 🎨 Particle Effects: Smooth animations and visual rewards
  • 💎 Real Rewards: Mint actual ERC-20 tokens from gameplay

🔗 Web3 Integration

Modern, type-safe Web3 development with the latest tools:

// Type-safe smart contract interactions import { useWriteContract, useWaitForTransactionReceipt } from 'wagmi'; import { parseEther } from 'viem'; const { writeContract } = useWriteContract(); // Mint gaming tokens writeContract({ address: CONTRACT_ADDRESS, abi: MRC_ABI, functionName: 'mineTokens', args: [parseEther('100')] });
